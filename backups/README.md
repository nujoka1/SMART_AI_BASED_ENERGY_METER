Backups: archived firmware sketches
=================================

This folder contains archived copies of duplicate Arduino sketches moved during
canonicalization. The project now keeps `smart_energy_monitor.ino` as the single
canonical firmware source. The archived files are retained for reference and
historical comparison — they are not compiled or used by default.

If you want to restore a sketch, copy it back to the project root or preferred
sketch folder and remove this archive copy.
